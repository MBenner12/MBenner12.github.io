Summary of Global Weather Plots



Visualizations



The purpose of this project was to dive deeper into how weather changes when you get closer to the equator. In order to do so, we pulled data from the OpenWeatherMap API which has a dataset of over 500 cities.


In Python, Matplotlib was used to plot various weather factors vs. latitude. The weather factors used were temperature, cloud cover, wind-speed, and humidity.


Then, these aspects of weather were plotted along longitude vs. latitude lines for comparison.
